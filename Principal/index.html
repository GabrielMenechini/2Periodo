Magic Age

Esse é um jogo via codigo, você fará todas as ações por comandos escritos logo abaixo, use os codigos corretamente para não ocasionar erros. Todas as palavras em negrito tem alguma interação com jogo, fique atento sobre isso.

Digite o comando

Comandos Básicos

  • /HELP Mostra os comandos disponiveis

  • /RESTART Reinicia o jogo, perdendo todo o save

  • /USE [ITEM] Interage com um item do cenario

  • /GET [Item] Caso esteja dentro das condições, adiciona o item no inventario

  • /INV Mostra seu inventario

  • /USE INV [ITEM] EM [ITEM CENARIO] Usa um item do inventario em algo do cenario

  • /SAVE Salva o jogo

  • /LOAD Carrega um jogo salvo
